
# ❌⭕ Tic-Tac-Toe

A classic **Tic-Tac-Toe** game playable in the browser, built with **HTML**, **CSS**, and **JavaScript**.

---

## 🚀 Live Demo

🔗 [View Live](https://amanyhisham.github.io/tic-tac-toe/)

---

## ✨ Features

- 🎮 Two-player game (X vs O)
- 🔵 Choose your symbol: X or O before starting
- 🏆 Win detection across all rows, columns, and diagonals
- 🔄 Reset Game button to restart at any time
 

---

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript

## 🧑‍💻 How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/amanyhisham/tic-tac-toe.git
   ```
2. Open `index.html` in your browser.
3. Choose **X** or **O** and start playing!
